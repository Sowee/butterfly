What is Butterfly Header ?

Butterfly header is an easy to use jquery modul that allows you to fix any element in the top of your html page. You can also choose between 3 type of animation (See options to learn more)

Donwload

Cross-Browser compatible

Butterfly header is Cross-browser and can be runned on any devices. It's build on responsive concept and you also can use it for an determinate size. (See options to learn more)

How to use it ?

Include JS Files


  <script src="butterfly.min.js"></script>

Minimal setup

Launch pluging using jQuery Selector


  $('.YourNav').butterfly();

That's all !

Example with default options

And you can also custom it with different options 


  $('.YourNav').butterfly({

      typeAnimation: 'hiddenAnimation',
      className: 'myClassCss',
      cssAnimation: true,
      transitionDuration: 0.6

  });

Options

Setting Default value Description
typeAnimation none Define animation when your element is fixed to top
className none Add a class when element is fixed top
cssAnimation true Use CSS3 animation for transition
transitionDuration 0.4 Define time of CSS3 transition
transitionDuration 0.4 Define time of CSS3 transition
